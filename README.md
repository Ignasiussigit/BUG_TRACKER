Aplikasi Bugtracker/helpdesk untuk Rumah Sakit berbasis website ini dibangun menggunakan bahasa pemrograman PHP dengan database MYSQL. code ini sudah bisa digunakan di PHP versi terbaru.
code ini sudah dilakukan ujicoba dan hasilnya berjalan dengan baik tidak terdapat eror atau tidak berjalan dengan semesterinya. Dalam pengkodean aplikasi helpdesk ini memang masih bantak yang perlu di kembangkan lagi dengan fitur-fitur terbaru sehingga akan menjadi semakin baik dan powerfull.

Fitur Aplikasi Helpdesk Berbasis WEB
Ada dua role user yang terlibat dalam aplikasi ini, role tersebut memilikit tugas masing-masing dan berbeda akses antara kedua role tersebut. Adapun pembagian fitur-fitur untuk tiap role adalah sebagai berikut :
Fitur Admin
•	Login
•	Halaman Dashboard
•	Kelola data admin
•	Kelola data teknisi
•	Kelola data Ruangan/Unit
•	kelola data user
•	Notifikasi Suara  tiket yang masuk
•	Proses penyelesaian tiket/pelaporan
•	Laporan tiket
•	gantipassword
Fitur Teknisi
•	Login
•	Dashboard Teknisi
•	Dhasboard Informasi total tiket, jumlah request, jumlah tiket proses dan jumlah tiket selesai.
•	Notifikasi Suara tiket yang masuk
•	Proses data tiket yang masuk
•	Balas perpesanan dengan yang mengajukan tiket (live)
•	Sudah autorefresh ketika ada data masuk.
•	ganti password

Adapun yang terakhir yaitu fitur user atau yang membuat laporan pengaduan.

•	dashboard user
•	kelola tiket saya
•	perpesanan / chat langsung dengan teknisi yang mengerjalan laporan (live)
•	sudah menggunakan summernote untuk pengajuan tiket
•	bisa menambahkan gambar
